A simple way to educate digitally.

Neutron is an application for windows-based whiteboards. With this software it is possible to educate in the time of digitisation without sparing one of the most important tools: The blackboard.

Neutron is not a crowded and cramped application, but is a user-friendly alternative to software like ActiveInspire or similar software products.

With Neutron, it is possible to give teachers a chance to educate in a very simple but modern way, without completely changing the way of teaching and sparing the most important tool. So it is nothing new, only getting back to the rules.

Neutron has been developed especially with teachers in mind that might have difficulties adjusting to electronic whiteboards. It has been developed by students and tested by their teachers. We took common frustrations from other programs as an inspiration

Functions.

  • Desktop and Browser.

    Neutron is built for modern browsers and as a desktop app.

  • Drawing & Painting.

    Easy writing and drawing on the whiteboard with the pencil tool.

  • Erase.

    Erase writings and drawings easily with the eraser tool.

  • Background.

    Change the background colour to every colour you want.

  • Saving.

    To keep track of your lessons, save the current board or export it as a PNG image file.

  • Open.

    Open saved boards so you can continue where you left off.

Additional Information.

Neutron is a browser based application and was also developed using Electron.

  • Electron desktop framework
  • online browser based
  • Secure no additional stored data
  • GitHub open source

Neutron is in development by the GitHub community and was initiated by Johann150. The aim of the application is being a simple, modern and user-friendly alterative to similar software like ActiveInspire by Promethan. With Neutron you can create a beautiful board in short time and with small effort and and at the same time you support the digitization of education. We as developers are aware that education software must always have a high level of privacy. This is why Neutron does not store any information, cookies, drawings or user data without you asking for it. "Asking for it" referring to the saving feature.

Help!

If something is not quite right.

I am using an operating system other than Windows.
It should be possible to overcome that problem. But the solution might get a bit techy. Please keep in mind that all steps taken by you are at your own risk.

  • First, you will have to install NodeJS. If there is no support for NodeJS on your operating system, we are sorry that we cannot help you out.
  • You will also have to download the neutron repository. For this you can either use git or just download a zip file from the repository website.
  • After finishing both of those and, if required, unpacking the zip file, you have to run npm run dist in the directory where you cloned/unziped to.
  • And you are done: There should be a file/folder created depending on your operating system.
