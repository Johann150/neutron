A simple way, to educate digital.

Neutron is a Application for windowsbased whiteboards. With this software it is possible to educate in the time of digitisation without sparing one of the most important tools: The Blackboard.


Neutron is not a crowded and cramped application, but is a user-friendly alternative to software like ActiveInspire or something else.


With Netron it is possible to give teachers a chance to educate in a very simple but modern way, without changing the complete way of teaching and sparing the most important tool. So it is nothing new, only getting back to the rules.

Functions.

  • Desktop and Browser.

    Neutron is built for Mozilla Firefox and as desktop app for Windows.

  • Drawing & Painting.

    Easy drawing and painting on the Whiteboard with the pencil-tool.

  • Erase

    Erase drawings and paintings easy and quickly with the erase-tool.

  • Background.

    To customize the whiteboarde change the backgroundcolor to every color you want.

  • Saving.

    So that you keep track of your lessons, save the panel paintig or export it to PNG.

  • Open.

    Open saved pictures so you can continue where you left off.

Additional Information.

Neutron is a browserbased application and was developed with Electron.

  • Electron Framework
  • Firefox Browserbased
  • Secure no stored data
  • Europe Austrian servers
  • GitHub Open Source

Neutron is in developement by the GitHub community especially by @Johann150 and @vonMitzscha. The aim of the application is being a simple, modern and user-firendly alterative to similar software like ActiveInspire by Promethan. So with Neutron you can create in short time and with small effort a beautiful panel painting and and at the same time you support the digitization of education. Developers are aware that education software must always be associated with a high level of privacy. So Neutron does not store any informations, cookies, paintings or user-datas.
